# Car Price Prediction using Machine Learning

## Project Description
This project predicts the selling price of a used car using Machine Learning. It uses the Random Forest Regressor algorithm to train the model and predict car prices based on different features.

## Features
- Load and clean the dataset
- Perform Exploratory Data Analysis (EDA)
- Visualize the data
- Train a Random Forest Regression model
- Evaluate model performance
- Predict the selling price of a new car

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Dataset
Dataset file: `car data.csv`

## How to Run
1. Install the required libraries:
   pip install -r requirements.txt

2. Run the program:
   python car_price_prediction.py

## Output
- Data cleaning
- Data visualization
- Model evaluation
- Predicted car price
