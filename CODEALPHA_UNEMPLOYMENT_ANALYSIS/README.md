# Unemployment Analysis with Python

## Objective

Analyze unemployment rate data using Python and visualize important trends through charts.

## Tools & Libraries

* Python
* Pandas
* Matplotlib
* Seaborn

## Features

* Data Loading and Cleaning
* Missing Value Analysis
* Statistical Summary
* Unemployment Rate Distribution (Histogram)
* Average Unemployment Rate by Region (Bar Chart)
* Correlation Analysis (Heatmap)

## Project Structure

CODEALPHA_UNEMPLOYMENT_ANALYSIS/

├── unemployment_analysis.py

├── Unemployment.csv

├── README.md

└── requirements.txt

## How to Run

1. Install the required libraries:

pip install -r requirements.txt

2. Run the program:

python unemployment_analysis.py

## Output

The program generates:

* Histogram of unemployment rates
* Bar chart of average unemployment rates by region
* Correlation heatmap

## Conclusion

This project analyzes unemployment data and presents meaningful insights using data visualization techniques in Python.
